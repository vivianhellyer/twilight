Build this into an embed.

pub fn color(self, color: u32) -> Result<Self, EmbedColorError>[src]

Set the color.

This must be a valid hexadecimal RGB value. 0x000000 is not an acceptable value as it would be thrown out by Discord.

Examples

Set the color of an embed to 0xfd69b3:

use twilight_embed_builder::EmbedBuilder;

let embed = EmbedBuilder::new()
    .color(0xfd_69_b3)?
    .description("a description")?
    .build();

Errors

Returns EmbedError::NotRgb if the provided color is not a valid RGB integer. Refer to COLOR_MAXIMUM to know what the maximum accepted value is.

Returns EmbedError::Zero if the provided color is 0, which is not an acceptable value.
